# Package index

## Data objects for XML versions of the SRPPP

- [`srppp_list`](https://agroscope-ch.github.io/srppphist/reference/srppp_list.md)
  : List of 'srppp_dm' objects for all years starting 2011
- [`srppp_active_substances`](https://agroscope-ch.github.io/srppphist/reference/srppp_active_substances.md)
  : Table of active substance names extracted from all years starting
  2011
- [`srppp_substances`](https://agroscope-ch.github.io/srppphist/reference/srppp_substances.md)
  : Table of substance names extracted from all years starting 2011
- [`srppp_products`](https://agroscope-ch.github.io/srppphist/reference/srppp_products.md)
  : Table of products extracted from all years starting 2011
- [`srppp_compositions`](https://agroscope-ch.github.io/srppphist/reference/srppp_compositions.md)
  : Table of the latest product compositions
- [`srppp_ingredients`](https://agroscope-ch.github.io/srppphist/reference/srppp_ingredients.md)
  : Table of product ingredients extracted from all years starting 2011
- [`srppp_pests`](https://agroscope-ch.github.io/srppphist/reference/srppp_pests.md)
  : Table of pesticide targets ("pests") extracted from all years
  starting 2011
- [`srppp_obligations_spe3`](https://agroscope-ch.github.io/srppphist/reference/srppp_obligations_spe3.md)
  : Table of all SPe 3 obligations and corresponding mitigation measures
- [`srppp_xml_dates`](https://agroscope-ch.github.io/srppphist/reference/srppp_xml_dates.md)
  : Publication dates of the available zip files
- [`srppp_xml_product_use_not_authorized()`](https://agroscope-ch.github.io/srppphist/reference/srppp_xml_product_use_not_authorized.md)
  : Check if a product name indicates that its use is not authorized at
  the time
- [`srppp_xml_zip_files`](https://agroscope-ch.github.io/srppphist/reference/srppp_xml_zip_files.md)
  : Relative paths of the available zip files

## Functions to work with the data

- [`product_categories()`](https://agroscope-ch.github.io/srppphist/reference/product_categories.md)
  : Get product categories for a pNbr
- [`composition()`](https://agroscope-ch.github.io/srppphist/reference/composition.md)
  : Retrieve one or more product compositions
- [`pNbr()`](https://agroscope-ch.github.io/srppphist/reference/pNbr.md)
  : Retrieve and select a P-Number for a product name
